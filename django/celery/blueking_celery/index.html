安装消息中间件

目前 celery 支持 redis、rabbitmq 作为任务的消息队列，推荐使用 redis。

mac 系统 redis 使用指南：

  • 安装指令 brew install redis；
  • 启动指令 redis-server；
  • 打开客户端 ./bin/redis-cli 。

windows 系统 redis 使用指南：

下载安装地址： https://github.com/MicrosoftArchive/redis/releases。

点击安装目录下的 redis-server.exe 启动 redis 服务。
