多进程

在 Unix/Linux 下，我们可以通过使用 fork()来实现多进程，fork()调用一次，返回两次，因为操作系统自动把当前进程（称为父进程）复制了一份（称为子进程），然后，分别在父进程和子进程内返回。子进程永远返回0，而父进程返回子进程的 ID。这样做的理由是，一个父进程可以 fork 出很多子进程，所以，父进程要记下每个子进程的 ID，而子进程只需要调用getppid()就可以拿到父进程的 ID。

Python3 迭代器与生成器

生成器

在 Python 中，一边循环一边计算的机制，称为生成器：generator。只需要将列表生成式的[]改成()，就生成了 generator，并且通过 next()函数获得 generator 的下一个返回值。

需要注意的是计算到最后一个元素，没有更多元素时，会抛出 StopIteration 的错误，所以一般使用 for 循环来迭代。

迭代器

可以被 next 函数调用并不断返回下一个值得对象称为迭代器：Iterator，生成器都是 Iterator 对象，但 list、dict、str 虽然是 Iterable，却不是 Iterator，使用 iter()函数则可以把它们变成 Iterator。

filter

Python 内建的 filter()函数用于过滤序列，filter 也接收一个函数和一个序列，filter 把传入的函数依次作用于每个元素，然后根据返回值是 True 还是 False 决定。

sorted

Python 内置的 sorted()函数可以对 list 进行排序。
