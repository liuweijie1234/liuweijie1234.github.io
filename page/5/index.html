检索和替换

re模块提供了re.sub用于替换字符串中的匹配项。

语法：

re.sub(pattern, repl, string, count=0, flags=0)

参数：

pattern : 正则中的模式字符串。
repl : 替换的字符串，也可为一个函数。
string : 要被查找替换的原始字符串。
count : 模式匹配后替换的最大次数，默认 0 表示替换所有的匹配。
flags : 编译时用的匹配模式，数字形式。

多线程

Python 提供了_thread 和 threading 两个模块来实现多线程，_thread 是低级模块，_threading 是高级模块，对_thread`进行了封装。绝大多数情况下，我们只需要使用 threading 这个高级模块。

多线程会造成数据的不安全，在 Python 中可以通过加锁来保证数据安全：

ThreadLocal

ThreadLocal 就是给每个线程创建一个全局变量的副本，这样每个线程都只能读写自己的副本，可以任意读写互不干扰，也不用进行锁的管理。
